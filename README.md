# Nmediaservices Single-Page Responsive Website

Welcome to the Nmediaservices Single-Page Responsive Website repository. This project is a modern, high-end editorial website built with React, Vite, and Tailwind CSS. It is designed to bridge the gap between high-precision engineering and avant-garde artistry, featuring intentional asymmetry, massive typographic contrast, and a dark-mode architecture.

## 🚀 Technologies

This project is built using:
- **[React 19](https://react.dev/)**: For building a dynamic and responsive UI.
- **[Vite 8](https://vitejs.dev/)**: Next-generation frontend tooling for rapid development.
- **[Tailwind CSS 4](https://tailwindcss.com/)**: A utility-first CSS framework for styling.
- **[React Router 7](https://reactrouter.com/)**: For client-side routing.

## 🎨 Design Philosophy

Our design system is built off the "Digital Architect" creative polar star:
- **Dark Mode First**: Sophisticated interplay of deep charcoals (`#121317`), stark whites, and subtle metallic greys.
- **Typography Engine**: Space Grotesk for massive, bold headings paired with Inter for clear, functional body text.
- **Structural Elegance**: We favor tonal layering over 1px borders, using "surface-container" hierarchies and glassmorphism (translucent backdrops and blurs) to achieve depth.
- **Fluid Motion**: All elevation changes feature smooth, snap-like animations (`cubic-bezier(0.16, 1, 0.3, 1)`) making the experience feel curated and expensive.

For a detailed breakdown of the UI tokens and structural rules, see [DESIGN.md](./DESIGN.md).

## 💻 Getting Started

### Prerequisites
- Node.js (v18 or newer recommended)
- npm or yarn

### Installation

1. Clone the repository to your local machine:
   ```bash
   git clone <your-repo-url>
   ```

2. Navigate into the project directory:
   ```bash
   cd Nmediaservices.com_Website
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the App Locally

To start the Vite development server with Hot Module Replacement (HMR):
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to see the app running.

### Building for Production

To create an optimized production build:
```bash
npm run build
```
The bundled files will be generated in the `dist` directory. To preview the production build locally, run:
```bash
npm run preview
```

## 📂 Project Structure

- `src/` - The main source code.
  - `pages/` - React components acting as individual pages (e.g., `HomeMobile.jsx`, `HomeDesktop.jsx`, `IPSeries.jsx`).
  - `App.jsx` - The main React application entry point containing routing logic.
- `Assets/` - Static images, fonts, and global design assets.
- `index.html` - The HTML template used by Vite.
